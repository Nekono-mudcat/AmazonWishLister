# Amazon Wishlister (amazonWishlister.ahk)

Amazonで割引されている商品を効率よく見つけるための自動化ツールです。

## 概要
このツールは、Amazonのページ内で特定の割引率（％オフ）が設定されている商品を自動的にリストアップ、または抽出するために作成しました。

## 主な機能
* **割引商品の自動検出:** 指定した条件に一致する割引アイテムを特定します。
* **ブラウザ自動操作:** 手動で探す手間を省き、効率的にセール情報をチェックします。
* ** NVDAなどのスクリーンリーダーでアクセシブル** 読み上げを前提とした設計。


## 開発環境 / 使用言語
* **AutoHotkey (AHK):** メインのスクリプト処理
* **JavaScript:** ブラウザ上のデータ抽出補助


## 使い方
1. AutoHotkey2.0以上をインストールした環境で `amazonWishlister.ahk` を実行します。
2. Amazonのウィッシュリストページを開いてCtrl + Endをしてページを最後まで読みこむ。
3. F8を押すとスキャン開始。
4. 日付の入ったテキストファイルを保存するように指示されるので保存。また、ヒットしたのかも読み上げるので適宜OKなどを押す。
5. 自動的に終了するが、緊急停止はEscape。
